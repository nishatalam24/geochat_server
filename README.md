# My MVC App

This is a simple MVC (Model-View-Controller) application structure.

## Project Structure

- **models/**: Contains model files that define the data structure and business logic of the application.
- **views/**: Holds view files responsible for rendering the user interface.
- **controllers/**: Includes controller files that handle the input and interaction between the models and views.
